# Shabba’s Bakery Website

## Overview
Shabba’s Bakery is a responsive website showcasing fresh bread, cakes, and pastries in Polokwane.  
It includes dynamic product listings, enquiry and contact forms, SEO optimization, and social media integration.

## Features
- Navigation across Home, About, Services, Enquiry, Contact
- Dynamic product loading with search filter
- Form validation for email and phone (072 319 2291)
- SEO meta tags, alt text, internal linking
- Scroll-to-top button
- Facebook and Pinterest icons labeled “Shabba’s Bakery”

## Deployment
- Hosted on Netlify or GitHub Pages
- Public link submitted via LMS
